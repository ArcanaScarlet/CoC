# 人物イラスト（推敲用）

**置き場**: `CoC/assets/人物/推敲用/`  
**公開用**（確定後）: `CoC/assets/人物/公開用/` — 推敲が固まるまで使わない

---

## 命名規則

```
{氏名}_v{番号}.{拡張子}
```

| 例 | 意味 |
|----|------|
| `東条育美_v1.jpg` | 東条育美の初稿 |
| `東条育美_v2.jpg` | v1 から修正した2稿目 |
| `番場吉蔵_v1.png` | 番場吉蔵の初稿 |

- **番号は1から**。修正のたびに **+1**（v1 → v2 → v3 …）
- **旧版は削除しない**（差分指示のベースに使う）
- 1キャラに複数ポーズが必要なら: `東条育美_v3_半身.jpg` のようにサフィックス可（番号はメインの世代）

---

## 構図・画風（作者指定・2026-06-18）

| 項目 | 指定 |
|------|------|
| カメラ | **目線の高さ**（正面〜やや斜め）。**上からのアングルは使わない** |
| ポーズ | 立ち絵（全身）。かわいめにするなら**斜め前**（3/4）が基本 |
| 背景 | **真っ白**のみ。風景・グラデ床・影絵は入れない |
| 用途 | 会話NPCの立ち絵・ブロック君／Cocofolia用のたたき台 |

---

## 作業の流れ

1. **新規** … `{氏名}_v1` を生成し、このフォルダに保存
2. **修正** … 指定ベース版を `image_edit` 等で改稿 → `{氏名}_v{N+1}` として保存
3. **台帳** … 各キャラの `*_イラスト台帳.md` に1行追記（下記）
4. **確定** … 作者OK後に `公開用/{氏名}_公開.png` へ別途書き出し（推敲用 v は残す）

---

## 作者からの修正指示の書き方（例）

ベース版を必ず指定すると伝わりやすい。

- 「**東条育美 v1** をベースに、帽子を博士帽にして **v2**」
- 「**v2** からローブをもっとHP寄りのシンプルに。背景は白のまま **v3**」
- 「**番場吉蔵 v1** はいいから表情差分だけ **v2**」

省略時は **そのキャラの最新 v** をベースにする。  
例外: **東条育美**は基本 **v10**（v14 は裾飾り参考のみ。最新 v ではない）。

---

## 台帳ファイル

キャラごとに同フォルダに `{氏名}_イラスト台帳.md` を置く。

| 列 | 内容 |
|----|------|
| 版 | v1, v2, … |
| ファイル | 実ファイル名 |
| 日付 | 作成日 |
| ベース | どの版から派生したか（v1 なら「新規」） |
| メモ | 参考元・指示内容・用途 |

設計メモ（`執筆・推敲用/人物_*_設計メモ.md`）の「イラスト」節からもこの台帳へリンクする。
