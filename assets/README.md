# assets / イラスト・画像置き場

このフォルダはシリーズ全体のイラスト・画像を管理するための場所です。
**3-tierの原則を厳守してください**。

## フォルダ構成
- `人物/公開用/`  
  PLハンドアウトや公開用資料に使える安全なイラストのみ。
  - 安心院教授の「大学教授・支援組織リーダー」としての通常の肖像
  - 久利那須教授の協力者らしい穏やかな姿
  - その他、プレイヤーが最初に見ても問題ないキャラ絵

- `人物/推敲用/`  
  執筆・推敲用のみ。秘密やホラー要素を含むバージョン。
  - 安心院のツァトゥグァ契約後・低SAN状態の姿
  - ショゴスロードのメイドを従えているバージョン
  - 洋館の魔法陣や神殿関連のコンセプトアート
  - ニャルラトホテプ関連のトラウマ表現（深層設定用）
  - 銀の黄昏マスターとしての本当の姿など

- `その他/`  
  マップ、魔法陣の図、アイテム、背景、モンスターのイラストなど。
  必要に応じてサブフォルダを作成（例: 魔法陣/、クリーチャー/）。

## リンクの書き方（Markdown / Obsidian）
相対パスで記述してください。

例（執筆・推敲用/人物_安心院教授_設計メモ.md から）:
```markdown
![安心院教授（公開用）](../../../assets/人物/公開用/安心院_通常.png)

**推敲用フルバージョン（秘匿）**
![安心院 朱璃 - ツァトゥグァ契約後](../../../assets/人物/推敲用/安心院_ツァトゥグァ.png)
```

公開用資料（PLハンドアウトなど）では**絶対に推敲用フォルダを参照しない**こと。

## 注意
- 画像ファイル名はわかりやすく（例: `安心院_教授_通常.png` / `安心院_秘密_ツァトゥグァ.png`）
- **Cocofolia対応**: キャラクター画像は必ず背景透過のPNG形式で保存してください。公開用は安全なもののみ、推敲用は秘密要素を含むバージョン。
- 生成したイラスト（AIなど）はここに保存。
- 将来的にシナリオをアップロード/公開する際は、`公開用`フォルダの中身だけを配布対象にできるようにする。
- シリーズ共通キャラ（安心院、久利那須など）はここに集約。シナリオごとの固有キャラは各シナリオフォルダの assets/ に置いてもOK。

生贄の神殿などの参考資料用イラストも、必要ならここにサブフォルダを作って管理してください（例: `生贄の神殿/参考/`）。

この構造は黄昏の失楽園シリーズの3-tier（PL / KP / 推敲用）を画像でも守るためのものです。
```
