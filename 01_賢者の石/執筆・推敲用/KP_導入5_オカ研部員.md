## ペア導入

### E＋A — 路傍の怪死に居合わせる（◎推奨）

| 項目 | 内容 |
|------|------|
| **主役** | ルートAのPC（スマホ・BadApple） |
| **Eの立場** | 大学帰りの**第二目撃者** |
| **時刻** | 5/24(月) **夕方**（`KP_導入1`と同時） |

**Eの読み上げ前フック**：

> 金田先輩のことが頭から離れない。御津門から一松方面へ帰る途中、人影の少ない路地で悲鳴がした。

**進行**：`KP_導入1_路傍の怪死.md`「ペア導入 E＋A」に従う。

**Eだけが持つ文脈**（口頭）：

- 数日前、LINEで金田先輩が「俺は賢者の石を手に入れる！」→ 翌日変死
- 「また同じような死に方……？」と直感
- BadAppleは**初聞き**でよい（店の知識は `KP_人物_黒須.md`）

**合流**：Aと**同行**でBadAppleへ。Eが「先輩のことも調べたい」と動機づけ。

### E＋D — 舞の依頼に同席する

| 項目 | 内容 |
|------|------|
| **主役** | ルートDのPC（舞・依頼・金田の入口） |
| **Eの立場** | オカ研**部員**として部室にいる |
| **向くD** | **D-2**（大学生）／**D-3**（舞の知り合いを部室に呼ぶ） |

**読み上げ**：

> オカルト研究会の部室。金田先輩の席が空いている。会長の舞が、信頼できる探索者（D）に金田の変死と調査依頼を話している。あなた（E）は部員として横にいる。

- 依頼の受諾・条件は**D**が決める。Eは感情・補足（「先輩、いつもあの席に……」）
- **D-1（探偵事務所）**ではE同席が不自然 → **E単独**か **E＋C** を検討

進行の芯 → `KP_導入4_舞からの依頼.md`

### E＋C — 5chで先輩を指認する

| 項目 | 内容 |
|------|------|
| **主役** | ルートCのPC（噂・5ch到達） |
| **Eの立場** | スレの **KND★** が金田先輩だと気づく部員 |
| **場所** | オカ研部室・大学PCルーム・カフェのノートPCなど |

**読み上げ**：

> オカ研の部室（または大学の共有PC）。隣の探索者（C）が「賢者の石」の掲示板を開いている。画面に **KND★** の書き込み——「行ってくる！」のあと、更新が止まっている。

**Eのセリフ例**：

> 「……あれ、うちの先輩だ。金田先輩、いつもオカ研で5chの話してた」

- 5ch・噂調査の**必須成果はC**が保持。Eは人間味・オカ研の空気
- Cがスレを読み進める間、Eが金田のエピソードを口頭追加可

進行 → `KP_導入3_噂の入口.md`

### E＋B — 一松で偶然出会う

| 項目 | 内容 |
|------|------|
| **主役** | ルートBのPC（被害1〜3） |
| **Eの立場** | 買い物・バイト帰りで聞き込み現場の**近く** |

**読み上げ**：

**Eのセリフ例**：

- 深く絡めないなら**短い遭遇**で切り上げ、別行動→BadApple合流でも可

進行 → `KP_導入2_事件概要.md`
