## シナリオ上の機能

```
探索者が図書館／禁書庫へ
  ├─ 通常利用 … 身分証・紹介状、または信用で受付通過 → 目録・資料案内
  ├─ 安心院ルート … 館長が鍵を渡す → 育美に同行を求める（原文）→ 地下へ
  ├─ 警備員ルート … 番場が鍵を持って同行 → 育美が一緒に来ると自然
  └─ 育美単独で禁書庫 … 鍵がなければ断る。「館長先生か番場さんと一緒なら」
```

**同行の範囲**

- 受付〜地下禁書庫の扉前〜庫内の**見回り確認**
- 「荒らし・持ち出し形跡の確認」が安心院から命じられた用件なら素直に協力
- **戦闘・危険対応はしない**。怖いものが出たら逃げる／「もう無理です」と離脱

**黄昏の失楽園（第1話）**

- メンバーだが**組織名・目的は説明しない**
- 探索者が黄昏の名を出しても「知ってる人？」程度。深入りさせない
- エンドの組織案内は安心院・舞の役。育美は同席してもよいが主導しない
