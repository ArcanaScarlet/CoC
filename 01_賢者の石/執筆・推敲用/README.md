## ファイルの見分け方

- `*_設計メモ.md` … 推敲・作者メモ（最優先で更新）
- `KP_*` … キーパー向けに書いた運用・台詞・技能値の整理（参照用。後で `KP_資料/` へ再書き出し可）
- `PL_*` … プレイヤーに見せる案（後で `PL_ハンドアウト/` へ再書き出し可）
- プレフィックスなしの `人物_*.md` 等 … キャラシート・事実データの塊

同名の「データ」と「KP_」は**内容が違う**ことがある。統合するときは設計メモを軸にマージすること。

## 人物イラスト（推敲用）

- **保存先**: `CoC/assets/人物/推敲用/`
- **命名**: `{氏名}_v1` → 修正のたび `_v2`, `_v3` …（旧版は残す）
