## 黄昏の失楽園における「ギルド窓口」

```
探索者・依頼人
      ↓ 夜、BadAppleへ
   黒須（受付）
      ├─ 依頼の紹介・仲介（土井事件など）
      ├─ 情報の価格表（価格・条件・待ち時間）
      ├─ 武器・マジックアイテムの取引
      └─ 名刺（リンゴ）→ ネットワーク内の協力者へ接続
      ↓
安心院教授（大学・学術ルート）とは別ハブ
```

- **安心院**: 昼・学園・神学・禁書庫の鍵（ダンブルドアー的）
- **黒須**: 夜・街・裏情報・武器（ギルド窓口＋裏社会）

PLは**基本 BadApple を知らない**。土井の遺言・検索・舞の案内・聞き込みで店を特定してから来店する——という導線。技能（酒・裏社会）で既知のPCは例外（`KP_人物_黒須.md`「探索者とBadApple」）。

## 性格・動機（表）

- **自分からは動かない**。バーにいて受け身。ギルド窓口——依頼は渡すが、現場には出ない。
- **情報の売買では探索者に友好的**。口調は乱暴でも、取引相手としては協力的で頼りになる。
- **お金にはシビア**。値引き・賒け・後払いは基本しない。無償は土井スマホ届けのお礼（合計30）など例外のみ。
- 探索者の実力を眼鏡で見極め、使える奴には情報を**売る**（同行投資ではない）。
- 土井のような仲間の死には素直に悲しむ（クロスの「ぶっきらぼうだが根は情に厚い」と接続）。
- 全部は話さない。**売る価値のない話はしない**タイプ。

### 第1話KP向け：所持金

BadApple購入が分岐に効くため、**PC所持金を開始前から記載・購入のたびに管理**する。お金がなければ情報も装備も手に入らない——黒須のシビアさとセットで運用。

## 裏設定（シリーズ全体・絶対秘匿）

**第1話（賢者の石）には関係しない。** `オリビニオスの囚人`（第3話）以降で効いてくる。**この節の内容をKP資料・データ・PL資料にコピーしない。**

### 経歴：元バチカンのエクソシスト

- 黒須は**もとバチカンのエクソシスト**である（D.Gray-manの「黒の教団エクソシスト」ではなく、**カトリック・バチカン側**の駆魔・異端審問の系譜）
- 現在は表向き**神父**（**黒須鞠庵**）として教会にいるが、それは二重生活の一面
- クロス・マリアン的な「科学者としての魔導研究」と矛盾しない——エクソシストとしての実務と、魔道・科学の素養を**並行して**持っている想定

### マギカロギア（魔法使い）との関係

- 黒須は**人間のまま**、**マギカロギア世界の魔法使い**との関係を持っている
- 魔法使い側の用語では、現世に干渉する魔法使いは**錨（アンカー）**となった人間と**運命共同体**で結ばれる（第3話原文：BadAppleでの説明）
- 黒須はその**アンカー**側。魔法使いの知識・危険性を理解しているが、**自分の立場について話すつもりはない**（第3話原文）

### マリアとの運命共同体（核心）

**マリア・アナスタシア・ローゼンクロイツ枢機卿**（第3話）の設定と直結する。

| 項目 | 内容 |
|------|------|
| マリアの正体 | 邪神の祝福により、**魔法使い**という不完全な生命として現世に呼び戻された**屍人（死人）** |
| 黒須の役割 | **アンカー**——死人であるマリアを、魔法使いとして**現世につなぎ留める** |
| 結びつき | **運命の力**で結ばれた運命共同体。黒須の力なくしてマリアは現世に留まれない |
| 秘密の共有者 | この事実を知るのは**黒須と安心院のみ**。決して口外されない |
| 一日のリズム | **日中**マリアの活動中は常に傍に控える → **日が沈み**マリアが棺で眠る → **夜**BadAppleで情報屋 |
| 情報源 | マリアに懺悔した者から記憶（罪悪感ごと）を奪い、黒須の情報網の一部にする |

### 元ネタ：グレイブ・オブ・マリア（D.Gray-man）

クロス・マリアンのイノセンス **『聖母ノ柩（グレイブ・オブ・マリア）』** がメカニズムのオマージュ元。

- 原作：すでに**死んでいる女性エクソシスト**の遺体を、クロスが**禁術で操作**し戦闘に用いる
- 本シリーズでの置き換え：
  - 「死んだエクソシストの遺体」→ **死人のマリア（屍人の魔法使い）**
  - 「禁術で遺体を操る」→ **アンカーとして現世に繋ぎ留める運命の力**（黒須が錨）
  - 戦闘時の「操って戦う」イメージはマリア側の戦闘力（書警相当）として第3話以降に表出。**断罪者・イノセンスそのものは登場しない**

※あくまで**設定の元ネタ・構造のオマージュ**。CoCシナリオ内にイノセンスや黒の教団は存在しない。

### 第1話との距離

- BadAppleのマスター・情報屋・リンゴマークの名刺——これだけが第1話に出る
- 神父・エクソシスト・マリア・アンカー・マギカロギアは**一切触れない**
- 第3話で「教会のマリアを紹介」「魔法使いとアンカーの話」をする**前提人物**として裏で置く
