## 解法：音楽で眠らせる

| 項目 | 内容 |
|------|------|
| **弱点** | **音楽を聞かせると眠る**（原文） |
| **現代版** | スマホ・タブレット・イヤホン＋スピーカー等で **曲を流す** |
| **成功** | 三つの頭が順に垂れ、いびき。入口へ進める |
| **失敗例** | 吠え続ける／片目だけ覚める。別の曲・音量で再試行可 |
| **時間** | 数ラウンド以内で処理。長考不要 |

## ヒントの入手経路

**どれか一つで十分**。複数重ねてもよい。

### 経路A：番場吉蔵のうっかりボロ（推奨・ハグリッド役）

番場はフラッフィーの世話をしている。**弱点を「攻略法」としては知らない**が、日常の愚痴として漏らす。

| きっかけ | 技能例 | 成功時に得られる情報 |
|----------|--------|----------------------|
| 時計塔・犬の話を振る | 説得／信用／心理学 | 下記台詞例 |
| 同行を頼んだあと | 同上 | より話しやすい |

**台詞例（そのまま使ってよい）**:

> 「あの子はなあ、三つ頭ぶん吠えるから困るんじゃ…。  
> 先日も若い学生がスマホで音楽をかけとったら、急に静かになってのう。  
> わしの耳がわるいんか、寝ておったんじゃよ。へへ、変な話じゃな」

- **意図**: ハグリッドと同型の「うっかり漏れ」。番場本人は気づいていない
- 失敗時: 「鍵は渡せんが、わしが付き合う」だけ。犬の話はしない

### 経路B：校内の聞き込み（運が良ければ）

| 技能 | 成功 | ハード／極限 |
|------|------|--------------|
| **聞き耳** | 「時計塔の裏にでけえ犬がいる」噂 | 「音楽をかけると寝るらしい」まで |
| **信用**（学生に） | 同上 | フラッフィーという通称 |
| **アイデア** | HP1を知っていれば連想 | そのまま音楽で試す動機 |

- 聞き込み場所: 図書館前、食堂、ゼミ棟、キャンパスベンチ等
- **運要素**: 通常成功でも「でかい犬」止まりでよい。深い情報はハード以上

### 経路C：PLの自力発見

| 方法 | 内容 |
|------|------|
| **現地実験** | 入口でスマホを鳴らして試す（ヒントなしでも可） |
| **アイデア** | HP1ファンが「フラッフィー＝音楽では？」 |
| **目星** | 犬小屋の近くに古いラジオ・CDプレイヤーの残骸等（任意の小物） |

**KP**: ヒントなしで音楽を試したPLには、そのまま成功してよい（原文どおりの弱点）。

## 警備員ルートとの接続

```
探索者が時計塔へ
  ├─ 番場に説得等で同行依頼
  ├─ 校舎裏へ案内 → フラッフィーが塞ぐ
  ├─ PLが音楽で眠らせる（番場は「ほう…」と驚く程度）
  └─ 番場が鍵を持って塔内へ → 最上階のチェス扉（別ギミック）
```

| 番場の態度 | メモ |
|------------|------|
| 犬の前 | 「わしはここで待つ」「先は若いのがなんとかしてくれ」 |
| 眠らせた後 | 「…まあ、通れそうじゃのう。ついてこい」 |
| 危険が見えたら | 従来どおり引き下がる（戦闘不参加） |

- フラッフィーは**戦闘モンスターにしない**（吠える・威嚇のみ）。寝かせるパズル
