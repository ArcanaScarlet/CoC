## 運用の要点

### 事件の見せ方

- PLには最初から全員の名前を渡さない。**導入2**なら1〜3、**BadApple**で土井、**現場**で鈴木、という段階開示が自然
- **死因（確定）**：**失血死**——体内の血液が急激に消失している。鑑識・医学では失血と判断される
- **不審死と呼ばれる理由**：**現場に血痕がない**（いつ・どこで失血したか不明）。搬送・不可視犯行の疑い。メディアは「吸血鬼事件」などと俗称する
- ホラー演出は星の吸血鬼（不可視）の気配で補強

### 儀式タイマー

| 日付 | イベント |
|------|----------|
| 5/20(木) | 魔導書（グラーキ12巻）盗難・5chスレ開始 |
| 5/21〜5/25 | 被害者1〜5（儀式準備が進行） |
| 5/24(月) | **シナリオ開始**（土井の日） |
| 5/26(水) | 満月・皆既月食・クライマックス |

被害が増えるたびに、KPは「街の空気が重くなる」「5chが荒れる」等の演出を足してよい。

## 被害者別メモ

### 1. 金田 博司

- **オカルト研究会メンバー**。**1人目の犠牲者**（確定 2026-06-30）
- 5ch ID **KND★**。隠しHPを見た直後に「行ってくる」
- LINE「俺は賢者の石を手に入れる！」→ 翌日変死（導入4・舞の依頼の芯）
- **御津門大学IP** — 御手神舞（導入4）・オカ研・導入5（E）と接続

### 2. 不知火 翔 / 3. 清水 幸雄

- 原文は詳細最小。警察資料・新聞クリッピングで肉付け可能
- 導入2の探索者が「まだ手がかりの薄い被害」として扱う

### 4. 土井 信数

- **黄昏の失楽園所属の探索者**（組織の顔ではないが仲間）
- 賢者の石の魔導書噂を追っていた。録画は残るが**犯人は映らない**
- スマホを届ける → **黒須**の依頼・無償情報（合計30）・名刺のきっかけ
- 黒須は「少し悲しそうな顔」——仲間の死を知っている描写

### 5. 鈴木 凡人

- **サラリーマン**。探索者の**目の前**で殺される演出向き
- 精神操作（イゴーロナク／星の吸血鬼の影響）で誘導
- スマホの赤霧市スレ → ネット調査・5ch再訪のフック
