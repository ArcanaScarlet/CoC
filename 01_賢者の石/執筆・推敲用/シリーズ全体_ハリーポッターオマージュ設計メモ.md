# シリーズ全体 ハリーポッターオマージュ設計メモ（執筆・推敲用）

**シリーズタイトル**: 黄昏の失楽園（全7作予定）
**全体のコンセプト**: ハリーポッターの各巻タイトルをパロディしつつ、クトゥルフ神話TRPGとして「楽園の黄昏（失われた理想・日常の崩壊）」をテーマに描く。
- PLが「あ、これあのシーン！」とクスッと笑えるようなオマージュ要素を散りばめつつ、すぐにホラーに捻じ曲げて不気味にする。
- 各話のタイトルは明確にHPパロディ：
  - 第1話: 黄昏の失楽園と賢者の石 （HP1）
  - 第2話: 黄昏の失楽園と秘密の部屋 （HP2）
  - 第3話: 黄昏の失楽園とオリビニオスの囚人 （HP3）
  - 第4話以降: 秘密の部屋、炎のゴブレット、フェニックスの騎士団、謎のプリンス、死の秘宝 のパロディを想定。
- 全体を通じて「学校（大学/赤霧市）を舞台にした禁断の力・取り憑き・儀式の連鎖」が進行し、最終的に「黄昏の失楽園」全体が崩壊する大団円へ。

## 各巻ごとのオマージュの方向性（随時追加）
- **第1話（賢者の石）**:
  - **過去事件（HP1の「ハリーの両親」＋ヴォルデモート破滅）**: **半世紀前〜それ以前**、イゴーロナク顕現で多数不審死 → 安心院ら探索者が結界を張り魔導書を地下に封じて「解決」。PLには「昔」の**あいまいな噂**のみ。第1話は**再来**（`世界観_赤霧市と御津門大学_裏設定.md`）。
  - 賢者の石の儀式 = 禁断の錬金術で「永遠の命や力」を得るが、代償は人間の心臓5つ（HPの「生贄」や「材料」パロディ）。
  - グラーキの黙示録を読むとイゴーロナク（悪意の精霊）に取りつかれる = クィレル先生にヴォルデモートが取りついているオマージュ。
  - **鏡の賢者の石**＝時計塔魔方陣の **ムナールの星石**（結界の核）。**探索者だけ**が外せる／イゴーロナクは自ら触れない。PLを「止めに来た」つもりで核除去させる魂胆（`ギミック_チェスと赤霧市_設計メモ.md`）。
    - イゴーロナク（**CoC公式邪神**）が「**第1話のラスボス**」。HP1の終盤でヴォルデモートが姿を現すように、クライマックスでイゴーロナクが顕現。**シリーズ全体の黒幕ではない**。
    - 取り憑かれた人物（久利那須教授）が「器」として機能し、儀式を進めようとする。
    - PLが「これクィレルだ！」と気づくような描写（頭の後ろに顔、奇妙な行動、汗をかいていない etc. をクトゥルフ的に捻る）。
  - 学校（御津門大学赤霧キャンパス） = ホグワーツの暗黒版。禁書庫、図書館、Hの間（隠された部屋/儀式空間）。
  - 5chリークやネット画像で現実が侵食 = 「秘密が漏れると災厄が広がる」現代版。
  - **鋼の錬金術師**（隠しHPの月食条項）: 皆既月食で月の影が街に落ち、円形の影＝魔方陣の外周、影が町に触れた瞬間に錬成発動——の連想を誘う **完全ミスリード**。「赤い石を退かせば止まる」も嘘（外すと罠）。詳細は `ギミック_チェスと赤霧市_設計メモ.md`。
  - 第1話の「イゴーロナク」はヴォルデモート**的**立ち位置（名を呼ぶと来る、知識が腐敗させる）。第2話以降は別の脅威が主役。

- **第2話（秘密の部屋）以降**:
  - 秘密の部屋 = 大学地下の隠された神殿や儀式場。蛇人間（バジリスク的クリーチャー？）や封印された存在。
  - 後の巻で「分霊箱」的な要素（心臓や本のページがイゴーロナクの欠片として機能）、対決の構造（学校対決、トーナメント、謎解き）をMythosホラーに変換。
  - 各話で「学校生活の理想（友情・成長・勝利）」が徐々に黄昏に染まり、クトゥルフ的な絶望（無力感、知識の恐怖、宇宙的無関心）に置き換わる。

## 全体を通じたオマージュのルール（推敲時のガイド）
- **クスッとさせる要素**:
  - タイトル直撃のシーン再現（例: 石を探す儀式、秘密の部屋探し、囚人脱走）。
  - キャラクターの行動やセリフで「HPのあのキャラっぽい」けど不気味（教授が「それは秘密の部屋だよ」と言う、呪文のような儀式がHPの呪文に似てる）。
  - 学校内の日常描写（授業、図書館、バーBadApple）をHPのホグワーツ生活に重ねつつ、怪異が忍び寄る。
- **すぐにホラーに変換するポイント**:
  - 友情・成長が「取り憑き」や「裏切り」に。
  - 勝利や「石」の力が「儀式の完成」で絶望的な結果に。
  - 魔法（賢者の石の儀式など）がクトゥルフの呪文や儀式に置き換わり、SANを削る。
- **シリーズ全体の進行**:
  - 第1話で「ヴォルデモート的イゴーロナク」の影を落とす（取り憑き開始）。※イゴーロナクは公式邪神。シリーズ黒幕ではなく第1話限定のオマージュ対象。
  - 各話で「分霊箱」的な要素（心臓、ページ、儀式の欠片）が散らばり、探索者が集めてしまう皮肉。
  - 最終巻（死の秘宝相当）で「死」や「永遠」をテーマに大決着。最大脅威は未定（イゴーロナクに限らない。銀の黄昏・他の旧支配者など複数候補）。
- **リスク管理**:
  - PLが「ただのHPパロディ」と油断するよう誘導し、後半で本気のMythosホラーをぶつける。
  - 感のいいPL向けに「悪意の精霊」などのぼかしを用意しつつ、気づいた人には「やっぱり！」と思わせる二重構造。
  - 各話の長さ（5-8時間想定）をHPの「1年の出来事」に重ね、徐々にエスカレート。

## 推敲時のチェックリスト
- このオマージュ要素が「クスッと」止まりでホラーを邪魔していないか？
- 各話のタイトルパロディがタイトル以外でも自然に織り込まれているか？
- シリーズ全体で「楽園の黄昏」が一貫して進行しているか（第1話で理想の崩壊の種を蒔く）。
- イゴーロナクの「取り憑き」がクィレル/ヴォルデモートっぽく、かつクトゥルフらしい（精神侵食、儀式加速、無力感）か。
- 後の話に布石を残せているか（心臓の欠片、グラーキの黙示録の影響、大学の秘密など）。
