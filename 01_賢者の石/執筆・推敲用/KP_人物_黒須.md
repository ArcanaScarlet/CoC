## シーン運用（BadApple初回）

1. **受け取り**: **ロックされた**スマホを受け取ったら奥へ。PLに待たせる。奥で**ロック解除**し、録画を確認（`KP_土井のスマホ録画.md`）。
2. **感情**: 再登場は悲しそうな顔——土井への友情を見せる（鬼畜キャラの人間味）。
3. **観察**: 踏み込んだら**眼鏡**で探索者を見る。信用・実力・度胸を測る演出。
4. **依頼**: 原文台詞で真相究明を依頼。報酬あり。
5. **お礼**: 合計価格**30まで**無償。PLが選ばせるか、KPが状況に応じて提案してよい。
6. **名刺**: 必ず渡す。以降の聞き込みで「黒須の紹介」として使える。

### RPのコツ（クロス・マリアン素）

- 敬語より**タメ口・命令調**（「〜しろ」「〜だろうが」）
- 煙草・仮面・赤髪は外見のアンカー。ギャグではなく威圧と風格
- 科学者っぽい突っ込み（「そんなもん知らん」「頭痛するなら見るな」）
- 借金・女の話は軽い毒舌として任意
- 土井の話のときだけトーンを落とす
- **自分から動かない**——「俺は店を開いてるだけだ。調べるのはお前らだろうが」系の突っ放し方でよい
- 金の話は曖昧にしない。「30までタダ。それ以降は値段表どおり」

## 所持金の管理（必須）

第1話では BadApple の購入が調査ルートに直結する。**PCの所持金をシナリオ開始前から記録し、購入のたびに減算する。**

| 運用 | 内容 |
|------|------|
| 事前準備 | キャラシートに所持金を記載（公開用整理版：「所持金は事前に記載しておいてください」） |
| 購入時 | 価格表の合計を確認 → 足りれば支払い・情報提供 → 足りなければ断る |
| 無償枠 | 土井スマホ届けのお礼のみ合計30まで（所持金を減らさない） |
| 高額例 | イゴーロナク(100)、星の吸血鬼(100)、イブン＝グハジ製造法(100)——複数買うと所持金が一気に減る |
| 黒須の台詞例 | 「金が足りねえなら帰れ」「タダは土井の礼までだ。それ以降は商売だ」 |

PLに「いくら持っているか」を常に意識させると、情報の優先順位付けがゲームになる。
