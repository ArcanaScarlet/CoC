**用途**: キーパーがセッション前〜本編中に参照する**入口**。資料の所在・開くタイミング・PL配布の区別をここで整理する。  
**正本フォルダ**: `執筆・推敲用/`（旧 `KP_資料/` は未更新の断片あり。迷ったらこちら）  
**原文**: `../原文/黄昏の失楽園と賢者の石.md`

---

## この目次の使い方

| いつ | 開く節 |
|------|--------|
| 初めて回す／資料が散らばって見える | **§1 全体像** → **§2 セッション0** |
| 導入の配役・進行 | **§3 導入** |
| 本編（BadApple以降） | **§4 調査** → **§5 クライマックス** |
| ファイル名だけ知りたい | **§7 ファイル一覧** |
| ネタバレ層・真相の整理 | **§6 KPが握る真相** |
| 執筆・未作成の把握 | `データ_ハンドアウト一覧.md` |

**凡例（状態）**

| 記号 | 意味 |
|------|------|
| ✅ | たたき台あり。回せる |
| 🟡 | 断片・設計メモのみ。KPが口頭補完 |
| ⬜ | 未作成 |
| 🖼 | 画像が別途必要 |

**資料の種別**

| 接頭辞 | 誰に | 備考 |
|--------|------|------|
| `KP_*` | **キーパーのみ** | 配布しない（導入台本・NPC運用・真相） |
| `PL_*` | **プレイヤー** | 調査成功・場面到達時に渡す |
| `*_設計メモ.md` | **作者→KP** | 意図・裏設定。PLにそのまま渡さない |
| 接頭辞なし `人物_*.md` 等 | **データ正本** | KP/PL用と内容が違うことがある |
