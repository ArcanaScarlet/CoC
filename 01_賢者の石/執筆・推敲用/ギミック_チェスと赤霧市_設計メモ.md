隠しHPの記述は **ダミーではないが、わざと曲げて書いてある**。  
作り方の骨格は **本当** —— PLが読み違えるよう **錬金術っぽい曖昧さ** で湾曲させている。

|--------------|----------|------------------------|--------------|------|

#### 「赤い石＋月明かり」条項 —— 鋼の錬金術師オマージュ（完全ミスリード）

隠しHPのこの段落は、**作り方として正しい部分と、意図的な嘘を混ぜた段落**。

**鋼の錬金術師へのオマージュ**（作者意図）:

- **皆既月食**で月の **影** を街に落とす
- 丸い月の影が魔方陣の **外側の円** になる
- **本影／半影** が町に落ちた瞬間に錬成が発動する——という連想をPLに持たせる
- → 「**街全体が魔法陣**。陣の中の人間が全員巻き込まれる」イメージ

**HPに書いてある「狙い」**:

| 文言 | PLへの効果 | 真偽 |
|------|------------|------|
| 中心に核となる **赤い石** | 「これが発動スイッチだ」 | 石は **ムナールの星石**（既存の結界核） |
| 月明かりが魔方陣を照らし **錬成発動** | 月食＝街規模の儀式開始 | 時計塔 **室内** の魔方陣が本番。街全体の陣は **読みの誤り** |
| （暗黙の誘導）赤い石を **退かす／外す** | 錬成を **止められる** と思う | **完全な嘘**（止まるどころか進む） |

**KPメモ**: PLが「赤い石を **退かせば** 錬成が止まる」と思うのが意図したミスリード。実際に外すと **罠**（結界無効化→本番）。正しい「止め方」は別途（退散呪文・星の吸血鬼撃破等）。

**時計塔での実際**（原文②③）:

- 皆既月食で月光が **弱くなる**（室内）
- 赤い **ムナールの星石** が魔方陣中央にある
- PLが石を **外す** → 緑に変色 → 久利「脅威除去」→ 本番
- のちに **別の** 赤い結晶（賢者の石完成品）ができる

**精製ギミックとの関係**（原文・賢者の石の製法）:

```
① 星の吸血鬼に人間の血を吸わせる → 代謝で水銀（純物質）を作る  …「複数の人間」＝血の分量
② 皆既日食に召喚（黒）
③ イブン＝グハジの粉（白）
④ 血を吸わせる（赤）
        ↓
時計塔の魔方陣で完成 → 賢者の石（赤い結晶）
```

- 被害5人は **材料の体** ではなく **血の供給源**（＋結界弱化の頂点）
- 久利の錬金術理解（硫黄と水銀の **純物質**）と矛盾しない
- PLが「5人殺された＝5人分の体が入る」と思うのは **意図した過読**

#### プレイヤーへのミスリード（**意図的・誘導用**）

上記の湾曲が、さらにこう読まれる:

> 賢者の石の錬成は **五つの頂点を持つ魔方陣** の中に材料を入れて行う

PLがここから組み立てやすい **誤解**（＝シナリオが望む読み）:

```
この街全体が「五頂点の魔方陣」
  ├─ 5人の被害＝頂点の生贄／材料
  ├─ 街の中の人間がみな材料にされる？
  └─ 中心（時計塔）で賢者の石の錬成が行われる
        ↓
「時計塔で儀式を止めなければ」
```

**KPが誘導したい行動**:  
五芒星の中心＝**時計塔**で賢者の石が作られようとしている → **そこへ向かって止める**。

- 行き先（時計塔）は **正しい**
- 動機の解釈（「街全体を釜にした錬成」）は **部分的に誤り** でよい
- 結界破壊の裏目的は、オカルト深入りやエンド後まで **全部明かさなくてよい**

- 久利は「PLを惑わす演出」だと信じている
- 実際はイゴーロナクの結界弱化と **意図がずれている**（本人も操られている）

### イゴーロナクの魂胆 —— プレイヤーを時計塔に呼び、核を外させる

**作者意図（2026-06-19）**: 五芒星・隠しHP・チェスパズルは、すべて **探索者を時計塔へ呼び寄せ、結界の核を自分の手で外させる** ための装置。

| 要素 | 内容 |

| **久利教授** | 取り憑かれているが、**自分では核を外す動機・演出がない**（むしろ儀式を進めたい） |

→ 邪神側は **「善人」役の探索者に取り除かせる** 必要がある。

PLが「自分は英雄側」と思っているほど、計画どおり核が外れる——という皮肉。


### C3：時計塔の扉（クライマックス①）

**オマージュ**: 『ハリー・ポッターと賢者の石』—— マクゴナガル先生が設置した **魔法のチェス**。駒と勝負し、**勝たないと先に進めない**。

本作では「本当にチェスを打ち切る」とセッションが止まるため、**フールズメイト（2手詰め）** で即決着させる。

#### 扉の仕様（原文）

| 要素 | 内容 |
|------|------|
| **場所** | 時計塔最上階。鋼鉄の扉。物理破壊不可 |
| **UI** | ディスプレイ ＋ キーボード `12345678abcdefghxPNBRQK+-#[START]` |
| **満月前** | 次の満月までのカウントダウン |
| **満月夜** | チェス盤が表示される |
| **[START]** | 白が自動で **Pf3**（f3）を指す |

#### フールズメイト（Fool's Mate）とは

チェス史上 **最短の詰み** の一つ。別名 **Two-Move Checkmate**（2手詰め）。  
白が初手から **愚かな** 指し方（f列・g列のポーンを無防備に前進）をすると、黒クイーンが斜め一撃で王を詰ます。

**標準手順**（代数式）:

```
1. f3   e5      ← 白がf3（扉はここまで自動）、黒がe5
2. g4?? Qh4#    ← 白の悪手g4の直後、黒Qh4で詰み
```

- 黒は **相手の悪手を待つ** 形。扉の「相手」は **フールズメイトの手を自動で打ってくる**
- プレイヤーは「このまま続ければ **即チェックメイト＝勝利**」と気づけばよい
- 知らなければ **ネット検索**（「フールズメイト」「2手詰め」）で調べられる——現代PL向けの正当な解法

#### 解き方（原文4系統＋整理）

| 解 | 方法 | 結果・描写 |
|----|------|------------|
| **1** | PLがフールズメイトに気づき、手を入力 | **Pe5** → **Pg4** → **Qh4#** で扉開錠（即時） |
| **1補** | 芸術(チェス)習得済み ＋ **アイデア**成功 | 1手目 **Pf3が不自然** と気づく |
| **1補²** | 上記アイデア **イクストリーム** | フールズメイトまで思い至る |
| **2** | **INT×1** または **芸術(チェス)** 成功 | 扉は開くが **長い対局** の描写（数分〜数十分かかった体） |
| **3** | 適切な装備 ＋ **鍵開け** 01クリティカル | 物理回避 |
| **4** | 適切な装備 ＋ **電気修理**＋**コンピューター** イクストリーム | システム迂回 |

**解1の流れ（KP向け）**:

```
[START]  → 盤面表示。白 Pf3（自動）
PL入力   → Pe5（黒の応手。フールズメイトの2手目）
PL入力   → Pg4（白の悪手——扉が求める「続き」）
PL入力   → Qh4#（黒の詰み＝「チェスに勝った」扱い → 扉が開く）
```

- 「勝つ」＝正規のフルゲームではなく **詰みの形を完成させる** こと
- 相手（黒）が e5 を打ってくるので、PLは **g4 → Qh4#** の気づきが核心
- 久利が後から追いつく場合: 探索者が扉を封鎖しようとすると「門の創造」系呪文で **内部から** 入り込む（原文）

#### C2との接続

| 経路 | 扉での体験 |
|------|------------|
| **5chで芸術(チェス)取得済み** | アイデアで Pf3 の不自然さに気づきやすい。技能成功なら解2の「長い勝負」も可 |
| **技能なし・チェス未経験** | 自分でフールズメイトに気づく／ネット調査／INT・鍵開け等 |
| **番地だけ先に解いた** | 「街もネットもチェスだった。塔の扉もか」—— オマージュの二重気づき |

#### HP1との対比（KP参考・PLに出さない）

| HP1・魔法のチェス | 本作・時計塔扉 |
|-------------------|----------------|
| 巨大な生きた駒と実戦 | キーボード入力の盤面UI |
| ロンが犠牲になり勝利 | フールズメイトで **数手で勝利** |
| 勝たないと先へ進めない | 同じ——**チェスで勝つ**ことが開錠条件 |
| 賢者の石へ続く最終関門 | ムナールの星石・賢者の石錬成へ続く最終関門 |
