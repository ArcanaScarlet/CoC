## 1. 赤霧市 — なぜこの街なのか

### 地理・雰囲気（原文＋作者補足）

- **東京都内**（西部）の架空区域・赤霧市。人口・産業は「大学と住宅街があれば回る」中小都市規模。
- **アーカム**がマサチューセッツにあるように、本作の「怪しい街」は日本版として赤霧市に置く（`ギミック_チェスと赤霧市_設計メモ.md`）。
- 市名の「赤霧」は正式な地名由来は**定かでない**（地方史では複数説）。表向きは夕焼け・工業時代の煤煙などの俗説。オカルト調査では「月が赤く見える夜が多い」「儀式の霧」などと脚色される。

### 龍脈（作者正本）

- 赤霧市一松地区の地下に**龍脈（レイライン）の合流点**がある。
- 御津門大学赤霧キャンパスは、その上に**意図的に**建てられた（後述）。
- 龍脈上では魔術・儀式の効力が増幅し、邪神・眷属の影響も入りやすい。**結界がないとキャンパスは「穴」になる**。

### 地方伝承 — 賢者の石と過去の噂（Cのみ・かなりあいまい）

| PLに出す噂（あいまい） | 内容 |
|------------------------|------|
| フラメル | 「どこかで読んだ」程度。**いつ・どこで**は定かでない |
| 錬成伝承 | **昔**、この辺りで賢者の石が作られた**かもしれない**という話 |
| 過去の変死 | **ずっと昔**、変な死があった**らしい**。いつ・何人・何が原因かは**誰もはっきり言えない** |
| （脚色） | オカ板の「人間を材料にした錬成じゃないか」——**推測の推測** |

**時期を特定させない。** 「十数年前」「○年前」といった数字はPL向けに使わない。  
**シナリオ中は上記の噂レベルにとどめる。事件の詳細は出さない**（後述§3・§8）。

| 作者正本（PLに出さない） | 内容 |

### 一松地区＝チェス盤（作者正本）

- 作者正本: 区画設計に関わった人物のうち、**銀の黄昏系の末端**が龍脈・五芒星の幾何を意識した（完全な魔術陣ではないが、**結果として**チェス盤と一致）。

## 2. 御津門大学 — なぜここに・何をしているか

### 表向き（C）

- 東京西部の**私立総合大学**。理学部に錬金術（化学史・思想史）ゼミがあるのが少し変わっている程度。
- 赤霧キャンパスは**図書館・時計塔**で有名。学外者は紹介状が必要。

### 裏の目的（A）

| 目的 | 由来 |
|------|------|
| **結界の維持** | **過去の魔導書事件**の封印を守る（時計塔・ムナールの星石） |
| **魔導書の封印保管** | 事件後の**禁書庫**——現在は**黄昏の失楽園管轄**（銀の黄昏は詳細不明） |
| **龍脈の監視** | 結界の効力が龍脈上で最大化するため、キャンパスが要衝 |
| **若手の観測・収集** | 事件後、安心院が**黄昏の失楽園**を創設。司書へ禁書庫管理を委託 |

大学は「学問の場」でありながら、**一度の惨事を封じ込めた墓標**でもある。

### 時計塔（A〜B）

- キャンパス**地理的中心**（一松4D＝d4）。龍脈の直上。
- 表向きは校舎のシンボル。内部上層に**満月までのカウントダウン装置**と**鋼鉄の扉**がある（原文クライマックス）。
- **隠し部屋**: 塔の根元〜地下に、五芒星魔法円と**ムナールの星石**がある。普段は探索者でも「ただの地下室」に見える。魔術的感知・図書館禁書で初めて「結界の核」とわかる。

### 年表（たたき台・A）

| 時期 | 出来事 |
|------|--------|
| 戦前〜戦後 | 御津門大学設立。赤霧市一松・龍脈上にキャンパス。 |
| 戦後 | 一松地区の区画整理（チェス状番地）。 |
| **半世紀前〜それ以前**（作者たたき台：1960年代頃） | **御津門・魔導書事件**（後述）。結界設置・禁書庫封印・事件「解決」。 |
| 事件後〜現在 | 安心院が結界・禁書庫を**維持**（館長就任はその後のいずれかの時点）。黄昏の失楽園創設。 |
| 第1話直前 | 久利がグラーキ12巻を持ち出し、憑依。**再来**の足掛かり。5chに写しを載せ、見た者の無意識を操って五頂点へ誘導するトラップ開始。 |
| 第1話（2021/5） | **吸血鬼事件**（再来の第一波）。PLが巻き込まれる。 |

※具体年は推敲で固定（例：2005年頃の事件 → 2021年の本編）。PLに一括で渡さない。

## 3. 過去の凄惨事件 — 結界が張られた理由（作者正本・確定 2026-06-28）

### HP1オマージュの骨格

| HP1 | 本作・過去事件 |
|-----|----------------|
| ヴォルデモートがハリーの両親を殺す | イゴーロナクが**正体に近い形**で顕現し、**多数**が不審死 |
| 赤ちゃんハリーに呪いが跳ね返り、ヴォルデモートは破滅 | 探索者たちが辛勝。イゴーロナクは**退けられたが消滅はしていない** |
| ハリーの傷／世間は「消えた」と信じる | 結界＋封印で**事件は解決した**と扱われる。街は忘れかけている |
| 第1話：クィレル経由で**再来** | 第1話：久利経由で**再来**（グラーキ12巻） |

「一度封じた悪が、また忍び寄る」構造を、シリーズ第1話全体の土台にする（`シリーズ全体_ハリーポッターオマージュ設計メモ.md`）。

### 事件名（作者用）

**御津門・魔導書事件**（通称：赤霧の連続不審死／表向きは「未解決の奇死事件」）

### 事件の流れ（作者正本）

```
① グラーキの黙示録（とくに12巻）が大学で読まれる／漏れる
        ↓
② イゴーロナクが顕現。司祭候補を求め、赤霧市一松を中心に暴走
        ↓
③ 大量の不審死（**作者正本：犠牲者は3桁規模＝数百人想定**。件数・名簿の具体は**恣意・あいまい**でよい）
   ・失血（干からび）、精神汚染
   ・目撃談：「口のある手」「頭のない白い巨体」（断片化・信憑性はばらつく）
        ↓
④ 探索者チームが対峙（**安心院朱璃を含む**）。チーム内で**名前が確定しているのは安心院のみ**——他メンバーは作者正本でも不明でよい。探索側にも犠牲・狂気者あり
        ↓
⑤ 龍脈上・時計塔下にムナールの星石を核とする結界を張る（星石は**当時の探索チームの一員が所持**していたもの）
   ・五頂点に印を刻み、半径1kmの古き印を完成
   ・イゴーロナクは「壁の向こう」へ／校内では力を出せない状態へ
        ↓
⑥ 魔導書類を図書館地下に移し、チェーンで封印（**禁書庫**の本格運用）
        ↓
⑦ 表向きの処理：風化・忘却。当時の報道も残りにくい（**半世紀前**のため）
   ・残るのは**口伝・断片的な噂**だけ。真実は探索者の間だけ
```

### 「解決」の意味

| 当時の認識 | 実際 |
|------------|------|
| 街は「怖い事件が終わった」と信じたい | イゴーロナクは**消滅していない** |
| 結界で校内は安全になった | 結界**外**では弱い侵食が残り得る |
| 魔導書を地下に封じれば再発しない | 12巻を読めば**再来の足掛かり**は残る |
| 安心院たちは勝った | 完全勝利ではなく**封印戦** |

第1話は、この封印が**意図的に削られ、再び破られようとしている**話。

### 二つの役割 — 封印の歴史とPLへのミスリード（作者正本・確定 2026-06-28）

| 役割 | 内容 |
|------|------|
| **裏の正本** | 結界・禁書庫は**イゴーロナク事件**の封印の名残 |
| **表の噂** | 街に残るのは「フラメル」「賢者の石」「昔の変死」など**時期不明の伝承**（口伝・ネットの創作が混ざる） |
| **PLへの狙い** | 「**過去にもこの町で人間を使った賢者の石の錬成があったのでは**」と思わせる**ダミー** |
| **現行事件との接続** | 隠しHP（生きた人間・五頂点）＋現在の5人被害＋過去の変死噂 → 錬成の**再演**に見える |

```
【作者が知っていること】過去＝イゴーロナク封印
【PLが聞く噂】          過去＝賢者の石・人間材料・フラメル・原因不明の変死
【PLが組み立てやすい誤解】「昔の錬成がまた始まった」
【第1話の真相】          再来はイゴーロナク。賢者の石は久利側の錬金術（別層）
```

**運用の鉄則**: シナリオ中、過去事件の**詳細は一切出さない**。

- 出してよい … **時期・件数・場所が定かでない噂**のみ（`PL_噂調査_賢者の石.md`、聞き込みの一言、5chの野次）
- 出さない … 年代の特定、イゴーロナク、グラーキ、安心院の当事者、封印儀式、ムナールの星石の由来、事件名、被害者名簿、当時の記録・詳報

NPCも「あの事件の真相」を語らない。安心院は**黙る／はぐらかす**。

### 安心院の立場（作者正本）

- 当時の探索者チームの**一員**（実質的な中心・封印儀式の執行者の一人）。
- 事件後、図書館長として**結界と禁書庫の維持責任**を引き受けた（「二度と起こさせない」）。
- 第1話では「あの事件のときもいた」と**明かさない**。PLには「ずっと前からある結界」と見える。
- 隠しHPの魔導書画像で強く反応するのは、**当時のトラウマの再燃**。
- 鍵を渡して探索者を追い出すのは、当時と同じく「知識に触れさせたくない」恐怖。

## 4. 結界（古き印）— 仕組みと維持

### 原文で確定していること

- 龍脈の合流点の**隠し部屋**に、**ムナールの星石**を中心とした**五芒星魔法円**。
- 半径 **1km** で、悪しきものの行動・精神支配・成功率を抑制（原文・安心院説明なし）。
- 円周上の **5か所** に**小さな五芒星**（**古き印の起点**）＝**過去の封印儀式**で刻まれた。
- エンド⑨: **古き印の結界**を再起動するにはムナールの星石＋**5人分のPOW**＋賢者の石（媒体）。

### なぜ張られたか — 一言で

**御津門・魔導書事件の直後、イゴーロナクを退け、二度と正体で街を歩かせないため。**

龍脈上だから効く、銀の黄昏の都合、などは**二次的要因**。一次理由は**惨事の封印**。

### 誰が張ったか（作者正本）

| 要素 | 設定 |
|------|------|
| **結界全体** | **安心院を含む探索者チーム**が事件の最終局面で設置 |
| **中心（ムナールの星石）** | 封印儀式の核。探索者側が用意／龍脈上で見つけた、など（詳細は推敲可） |
| **五頂点の印** | 同じ儀式で市内5か所に刻んだ**古き印の起点**（後から削られると弱まる） |
| **現代の維持** | 安心院が館長として継続監視 |
| **久利の認識** | 結界の意味を**知らない**。五芒星を「演出のダミー」と誤解 |

### ムナールの星石（A）

- **正体**: 古き印結界の**核**。赤い結晶。賢者の石の完成品とは別。
- **名称の由来**: ムナール（Mnar）系の伝承に由来する封印石、と図書館の禁書目録に1行ある程度（B6で開示可）。
- **PLがやりがちな誤解**: 隠しHPの「中心の赤い石」＝これだと思い込む → **意図したミスリード**（`ギミック_チェスと赤霧市_設計メモ.md`）。
- **外したとき**: 結界が切れる。イゴーロナク側の本望。エンドで再起動するのは**別の儀式**（⑨）。

### 結界と五芒星パズルの関係（整理）

```
【古き印・過去】          【第1話・現在】
五頂点に小さな五芒星  ←→  被害5地点の逆五芒星（結界を削る）
中心にムナールの星石  ←→  時計塔魔方陣（イゴーが外させたい）
一松の番地形状        ←→  PLを中心へ誘導するパズル（幾何の一致）
```

同一の星形構造の**表（過去の封印）と裏（第1話の破り）**として扱う（`ギミック_チェスと赤霧市_設計メモ.md` と整合）。

## 5. 禁書庫 — なぜ・誰の・どう管理するか

### 表向きの理由（B〜C）

- **神学・思想史・比較宗教学**の研究資料として、危険な写本を「封印保管」している。
- 安心院館長の方針:「知るべきでない知識も、**無管理で街に漏れるよりマシ**」。
- 学外者には原則非公開。鍵は**図書館長**と**警備員室**にもある（大学の規程上）。

### 裏の理由（A）— 一次理由

**御津門・魔導書事件の「解決」の一環として、魔導書を地下に封じた。**

- 事件の触媒だった文献（**グラーキの黙示録**ほか）を、もう街に放ち出せない。
- 図書館地下を**禁書庫**とし、チェーン・目録・立入制限を整備したのは**事件直後**。
- 設置したのは**当時の安心院と、その探索者仲間**（のちの黄昏の失楽園の中核）。

### 管轄（作者正本・確定 2026-06-28）

| 層 | 内容 |
|----|------|
| **設置** | 過去事件の当事者たち（**安心院＋仲間**） |
| **現在の管轄** | **黄昏の失楽園**（組織としての封印施設。表向きは大学図書館の一部） |
| **銀の黄昏** | 禁書庫の**詳細は知られていない**。存在の片鱗は大学関係者レベルで漏れても、中身・封印の理由・グラーキの扱いまでは上位も把握していない |
| **大学当局** | 「危険資料の保管庫」程度の認識。魔術的封印の実態は知らない |

安心院は銀の黄昏のマスターだが、禁書庫は**黄昏の失楽園側の私物**として切り分けている。アンシャトレーヌ派・久利ルートは、この管轄の**外**から割り込んだ侵害。

### 現在の管理（作者正本）

| 役割 | 担当 |
|------|------|
| **総括** | **安心院朱璃**（館長） |
| **実務・監視** | 管理を**委託された司書**（第1話：**東条育美**ほか。黄昏の失楽園の信頼できるメンバー） |
| **同行・立入制限** | 司書が地下へ**同行・監視**（触るな・持ち出すな） |
| **鍵（警備員室）** | **番場吉蔵**——大学の規程上、建物の鍵は警備に預ける必要があるため**仕方なく** |

### 鍵まわり（作者正本）

安心院の本音:**警備員室に鍵を置きたくなかった**。

| 鍵の所在 | 安心院の考え |
|----------|--------------|
| **館長（安心院）** | 正規の管理者。ここだけが理想 |
| **警備員室（番場）** | **大学の都合**で複製がある。防災・夜間巡回・建物管理の規程。魔導書の内容は番場には教えない |
| **司書（東条）** | **鍵は持たない**（原文どおり）。館長または警備員とセットで地下へ |

番場は「地下に危険な本がある」程度は知っていても、**グラーキ・イゴーロナク・封印の理由**は知らない。鍵を渡す／同行するのは**規則と上司の指示**の範囲。

### 裏の理由（A）— 二次理由

- 事件以前から大学には危険文献があった可能性はある（収集品・研究資料）。
- 事件を機に**一括で地下へ集約・封印**し、**黄昏の失楽園が実質管轄**した。

**チェーン・目録・同行制**は第1話時点での**最後の防波堤**。久利の持ち出しは管轄の**突破**＝事件の再発。

### 禁書庫に何があるか（方針）

| 種類 | 第1話での扱い |
|------|----------------|
| グラーキの黙示録（12巻体制） | **本話の核心**。12巻のみ欠落 |

## 6. グラーキの黙示録 — なぜ禁書庫に・なぜ12巻が危険か

### 公式との対応（A）

- 原典の『グラーキの黙示録』は**グラーキ（旧支配者）**系の魔導書。全11巻＋**夢に導かれて書かれた第12巻**がイゴーロナク接触の鍵（`人物_イゴーロナク_設計メモ.md`）。
- 本作では**12ページ／12巻**として調査進行を管理（儀式・被害イベントの段階化）。

### なぜ御津門大学にあるか（作者正本）

1. **過去事件の触媒**だった。事件後に地下禁書庫へ**封印**された。
2. 1〜11巻はグラーキ系の写本。**12巻（手書き増補）**がイゴーロナク接触の鍵——**当時もこれが引き金**。
3. 事件「解決」後、安心院たちは「12巻は特に危険」と知っている。チェーン・目録で管理。
4. 第1話：久利が12巻だけ持ち出す＝**封印の核心が再び外れた**。

### なぜ久利が読めたか（A）

- 久利は銀の黄昏**末端**だが、禁書庫の存在は**銀の黄昏上層にも詳しく知られていない**。
- アンシャトレーヌ派の**別ルート**（研究口実・大学内の抜け道・久利個人の執着）で接触した侵害。
- 表向きは正当な研究者。「錬金術史の調査」等の**大学内の顔**で近づいた可能性。
- **取りつき後**は画像を見ても頭痛しない（原文）。他者は見るだけでも汚染。

### 5chとの関係（作者正本・2026-06-28方針）

- イゴーロナク側が**12巻のページ写し**を5chに載せ、精神汚染と誘導を行う。
- **書名・表紙は写さない**（PLは「出典不明の英文原稿」として受け取る）。
- 書名**グラーキの黙示録**は**禁書庫目録・安心院の鑑定**で後から判明（`KP_5chスレッドと隠しHP.md`）。

### 盗難の意味（A）

- 5/20前後: 12巻が目録上なくなる（原文）。
- 久利が持ち出し、校内で読み取り憑き。残りのページは5ch・儀式に利用。
- 安心院は「荒らし・欠落の確認」で鍵を渡すが、**犯人が久利だと第1話中は言わない**（気づいているが追い詰めない）。
