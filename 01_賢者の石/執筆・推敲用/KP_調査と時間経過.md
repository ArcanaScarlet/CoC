# 調査と時間経過（KP用）

**用途**: 本編の移動・調査にかかる時間の共通ルール。  
**正本**: `原文/黄昏の失楽園と賢者の石.md`（噂調査2時間・BadApple 20〜24時）  
**関連**: `KP_目次.md` §4、`TALTO_冊子2_KP本編_目次.md`

---

## 基本ルール（作者指定）

| 行動 | 所要時間 | 備考 |
|------|----------|------|
| **移動**（調査箇所間） | **30分** | 大学↔警察署↔一松現場など、**エリアをまたぐ**移動は一律 |
| **調査**（1箇所） | **1時間** | 聞き込み・現場検証・施設内の聞き取りなど、**1地点1回**の目安 |

### 1「調査箇所」の数え方

| 算入 | 例 |
|------|-----|
| **1箇所＝1時間** | 警察署で聞き込み／一松2Bの現場／大学・図書館受付／警備員室／オカ研部室／久利研究室／安心院との面談／キャンパス内の学生聞き込み（1回） |
| **別箇所** | 図書館と警備員室は**別**。被害者ごとの現場も**別** |
| **移動のみ** | 上記のあいだに**30分**を挟む |

### 大学内の扱い

- **図書館**（東条・受付・目録）は大学エリアの**1箇所**
- **禁書庫**へ地下に降りる同行は、鍵・条件が揃ったうえで**別途1時間**（図書館調査とは別カウント可）
- **キャンパス聞き込み**（学生・教員へ声をかける）は**1回1時間**。対象を変えて続けてもよいが、追加1時間ごとに延長

## 施設別 — いつ行けるか

### 警察署

| 項目 | 内容 |
|------|------|
| 調査時間 | **1時間**／回 |
| 日付 | 5/24〜5/26 |
| 正本 | `KP_導入2_事件概要.md`（導入2）、原文「警察署で交渉系技能」 |
| 技能 | 信用・説得・交渉・聞き耳等 |
| 得られるもの | 被害者1〜5の段階的情報、公式見解（曖昧）、捜査の遅れ |

導入2以外のPCも、**身分・説得**が通れば聞き込み可。警官PCは追加情報を早く得てよい。

| 清水 | 6A | 5/24〜 |
| 土井 | 2F | 5/24〜 |

各現場 **1時間＋移動30分**。`KP_赤霧市番地と五芒星.md` 参照。

### 御津門大学

| 施設・人物 | 調査時間 | 主な利用可能日 | 備考 |
|------------|----------|----------------|------|
| **図書館**（東条・受付） | 1時間 | 5/24〜5/26 | 目録・一般利用。館長不在日も東条が窓口 |
| **警備員室**（番場） | 1時間 | 5/24〜5/26 | 禁書庫鍵・同行 |
| **オカ研**（舞） | 1時間 | 5/24〜5/26 | 導入4済みなら再訪可 |
| **久利教授** | 1時間 | 5/24〜5/26 | 錬金術ゼミ・研究室 |
| **安心院教授** | 1時間 | **5/24(月)・5/26(水)のみ** | **月水金出勤**。5/25(火)は**不在** |
| **学生聞き込み** | 1時間／回 | 5/24〜5/26 | キャンパス内。金田・噂・目撃など |
| **禁書庫** | 1時間 | 鍵・条件後 | 東条同行。書名判明は目録調査から |

**5/26（水）の大学**: 安心院に会える**唯一の調査日が残っている日**（5/24も月で可、5/25は火で不可）。禁書庫鍵・神学の説明を急ぐPC向け。

## 日付別スケジュール

### 5/24（月）— シナリオ開始

| 時間帯 | 内容 |
|--------|------|
| 昼〜夕 | 導入B・C・D・E（各PC） |
| 夕方 | 導入A（路傍の怪死） |
| **調査可能** | 導入と並行または導入後。**安心院＝本日出勤** |
| 20:00〜 | **BadApple合流**（推奨） |

### 5/25（火）— 鈴木の日

| 時間帯 | 内容 |
|--------|------|
| 昼〜夕 | **通常調査**（警察署・大学・現場） |
| **必須** | **鈴木・眼前殺害**（時間はKP裁量。調査1〜2箇分分を消費してよい） |
| **安心院** | **不在**（火曜） |
| 20:00〜 | BadApple（継続調査） |

### 5/26（水）— クライマックスの日

| 時間帯 | 内容 |
|--------|------|
| **昼〜夕** | **調査継続可**（警察署・現場・大学・学生聞き込み） |
| **昼〜夕** | **安心院教授に会える**（水曜出勤）— 急ぎの聞き取り・禁書庫鍵 |
| 17:00頃〜 | **時計塔・クライマックス**（`冊子2_KP向け情報.md` 月食表） |
| 20:00〜 | BadAppleは営業中だが、クライマックスと前後する場合あり |

**KP運用**: 5/26昼に「あと1箇所だけ大学へ」のような**最後の調査枠**を残しておくと、安心院ルートが自然。

## 1日に回れる目安

| 計算 | 目安 |
|------|------|
| 移動30分＋調査1時間 | **1.5時間／箇所** |
| 昼の調査可能時間 | **約8時間**（9:00〜17:00想定） |
| 1日あたりの箇所数 | **4〜5箇所**（移動・ロールプレイ・休憩込み） |

ネット2時間・BadApple1時間を使う日は、**現場調査は2〜3箇所**に抑える。

### 例（5/25の午後）

```
13:00  警察署で聞き込み（1h）→ 13:00完了
13:30  移動
14:00  大学・図書館（1h）→ 15:00完了
15:30  移動
16:00  一松6G・現場（1h）→ 17:00完了
夕方   鈴木イベント（必須）
20:00  BadApple
```
