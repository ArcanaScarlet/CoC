## 見出しの書き方（TALTO流し込み用）

TALTOは **H1＝章タイトル**、**H2＝項目**（左目次に出る）、**H3＝小項目**（目次に出ない）として自動表示される。

本ファイルでは、流し込み時にレベルが一目で分かるよう **見出し行の先頭に `H1：` `H2：` `H3：` を付ける**。

| 接頭辞 | TALTOでの扱い | このファイルでの markdown |
|--------|---------------|---------------------------|
| **H1：** | 大見出し（章） | `# H1：…` |
| **H2：** | 中見出し（項目・場面） | `## H2：…` |
| **H3：** | 小見出し（補足） | `### H3：…` |

**流し込み手順**

1. `H1：` で始まる行 → TALTOで **見出し1**
2. `H2：` で始まる行 → TALTOで **見出し2**
3. `H3：` で始まる行 → TALTOで **見出し3**
4. 投稿時は表示テキストから **`H1：` 等の接頭辞を削除**してよい（レベルだけ合わせる）

**KPの読み方**

```
【プレイ前】  H1：プレイ前に読む まで一通り
【5/24導入】  H1：導入パート ＋ 割り当てた H2 だけ精読
【合流以降】  調査・クライマックスの H2 を場面ごとにジャンプ
【参照】      NPC・表・ハンドアウト全文はデータ集
```
