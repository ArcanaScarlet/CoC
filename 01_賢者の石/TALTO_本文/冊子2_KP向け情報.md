# ストーリー概要

物語は、御津門大学の図書館地下──禁書庫──から始まる。

秘密結社銀の黄昏の末端である久利那須教授は、禁書庫に忍び込み、**グラーキの黙示録**十二巻を読み、持ち出す。その瞬間、**イゴーロナク**に憑依される。賢者の石を自分の手で作りたい──星の吸血鬼と人間の血、皆既日食の下での錬成。読んだ魔導書は、その手順を与えたように思える。

半世紀ほど前、同じ大学でも魔導書を巡る凄惨な事件があり、そのとき張られた結界が今もキャンパスを守っている。結界の核は時計塔の下にある**ムナールの星石**だ。校内では、あの事件の主たる存在──**イゴーロナク**──は力を出せない。

イゴーロナクは憑依したものの、校内では完全な顕現が結界に阻まれる。だから、ネットに魔導書の写しを漏らす。見た者の**無意識を操り**、結界の**五つの頂点**──古き印の起点──へ誘い出し、**星の吸血鬼**に襲わせる。吸われた血は賢者の石の材料になる。頂点を**死で汚す**たびに、結界は弱まっていく。久利は賢者の石を作るための実験だと思っている。イゴーロナクは、壁の向こうから自由になりたいだけだ。二人の思惑は重なるが、目指す先は違う。

その結果が、2021年5月の赤霧市にある。不審死が続き、マスコミは「吸血鬼事件」と呼ぶ。オカルト研究会の金田博司も掲示板の罠に踏み、スレに「行ってくる」と書いた直後、意識のないまま赤霧市一松の頂点へ向かい、姿を消す。続けて死者が出る。探索者たちは、路地での出来事、不審死の調査、掲示板、大学からの依頼など、ばらばらの入口から巻き込まれていく。

5月24日の夕方、黄昏の失楽園の探索者・土井信数が路地で倒れ、スマホをバー「BadApple」へ届けるよう遺す。情報屋の黒須の店で探索者は顔を合わせ、断片を照らし合わせ始める。

調べれば調べるほど、見えてくるのはこういう絵図だ。赤霧市一松の番地に刻まれた**古き印**が五つの頂点を示し、町全体を覆う五芒星の魔法陣のように見える。隠しHPや噂が示すのは、**人間を材料にした賢者の石の錬成**──**皆既日食が町を覆うとき**、その魔法陣が動く──という条件つきの話。昔の変死の話と重なり、「また同じ儀式が進んでいる」と読める**ダミー情報**。久利の実験や結界を破ろうとする何かの正体は、ここでは見えない。

探索者は自然と**時計塔**へ向かい、錬成を邪魔しようと動く。それが意図された誘導だ。中心の赤い石を外せば止まると思わせる罠に、自ら踏み込むかもしれない。

5月25日、鈴木凡人が探索者の眼前で殺される。翌26日、皆既日食が町を覆う夜──探索者が信じる期限どおり、儀式は時計塔で本番になる。探索者が星石を外してしまえば、久利は「脅威は取り除かれた」と言い、**星の吸血鬼**を呼び出して賢者の石の錬成を始める。久利を倒すと、首のない巨人──**イゴーロナク**──が顕現する。**イゴーロナク**が完全に顕現すれば、町はひとたまりもなくなる。

この事件を通じて、探索者は超常の裏に顔を出す者たちの往来──**黄昏の失楽園**──に触れる。シリーズの第1話。

## 5/26(水) クライマックス ─ 皆既月食

PLが信じるのは「**皆既日食**が町を覆う夜」。**実際の天文イベントは皆既月食**（満月の夜）。時刻は2021年5月26日・東京想定のたたき台。原文クライマックス②〜⑨と対応。

| 時刻（目安） | 天文・演出 | シナリオ |
|--------------|------------|----------|
| 昼〜夕 | — | 安心院教授：水曜出勤 |
| 17:00頃 | 満月の夜 | 時計塔・鋼鉄の扉のカウントダウン終了。チェスで開錠（①） |
| 18:45頃 | 月食・偏食の始まり | ②最上階の部屋に入室。鉄格子の窓から月明かり。光が弱まり始める |
| 19:09頃 | **皆既月食・入り** | 差し込む月光がさらに衰える |
| 19:18頃 | 食の最大 | 探索者が星石を外さなければ、汚染PCは毎R POWロール（勝手に外そうとする） |
| 19:28頃 | 皆既月食・終わり | ③ムナールの星石を除去 → 久利登場「脅威は取り除かれた」。**月光が完全に消える** |
| 直後〜 | 室内は暗闇・温度上昇 | ④星の吸血鬼顕現（イブン＝グハジの粉）→ 吸血で賢者の石の錬成が進む |
| 続く | — | 賢者の石完成 → 完全体久利 → 久利を倒すと⑦イゴーロナク顕現 → 退散呪文・結界再起動（⑨） |
