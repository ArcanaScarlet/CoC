# TALTO 本文（成果物）

[TALTO](https://talto.cc/) へ投稿する**最終原稿**を置くフォルダ。  
執筆・推敲は `../執筆・推敲用/`。内容が固まったらここへ書き出す。

## 冊子の役割

| 冊子 | ファイル | 誰が・いつ |
|------|----------|------------|
| **1** PL向け情報 | `冊子1_PL向け情報.md` | **プレイ前**。あらすじ・人数・技能・注意 |
| **2** KP本編 | `冊子2_KP向け情報.md`（現状）／今後・調査・クライマックス本文 | **プレイ前〜進行中**の投稿用原稿 |
| **3** KPデータ集 | （未作成） | **参照用**。NPC・表・ハンドアウト全文・クリーチャー |

## 作業用（推敲用フォルダ）

| ファイル | 用途 |
|----------|------|
| `../執筆・推敲用/TALTO_冊子2_KP本編_目次.md` | 冊子2の **H1/H2 設計図**（見出しは `H1：` `H2：` `H3：` 接頭辞付き） |
| `../執筆・推敲用/TALTO投稿用_構成案.md` | 冊子全体の外郭・投稿手順 |

TALTO投稿時は、目次ファイルの構成に沿って本フォルダへ流し込む。
