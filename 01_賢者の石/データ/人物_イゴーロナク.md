## 行動原理（原文ベース・KP用要約）

### きっかけ：たまたまの取りつき

久利那須教授が禁書庫で**グラーキの黙示録（手書き手稿12巻）**を読んだ結果、イゴーロナクに取りつかれる。  
イゴーロナクが久利を最初から「選んだ」という記述は原文にない。**偶然の接触が起点**。

### 最大の障害：御津門大学の退魔結界

龍脈の集まる隠し部屋に**ムナールの星石**を中心とした五芒星魔法円があり、半径1kmで悪しきものを阻害する。

| 結界内の効果 |
|--------------|
| 邪神の眷属は**移動できない** |
| 邪神による**精神支配が弱まる** |
| 邪神の行動**成功率が半分**になる |

このため、校内ではイゴーロナクは**人間を完全乗っ取りできない**。悪意を通じた間接操作に限られる。

### 真の目的

**結界を取り除き（無力化し）、完全な姿で顕現すること。**

賢者の石の錬成は、久利教授という「錬金術に取り憑かれた人間」を動かすための**餌（動機づけ）**でもある。久利本人は賢者の石づくりに固執しており、イゴーロナクはその欲求を利用して校内の結界突破と儀式進行を進める。

### 手段の全体像

```
取りつき（偶然）
  ↓ 結界で完全支配不可
悪意で久利教授を操る
  ├─ 5ch・噂で「愚かな人間」を集める（材料＋精神汚染）
  ├─ 結界外の星の吸血鬼で吸血殺人（吸血鬼事件）
  ├─ 逆五芒星で結界を弱める
  ├─ 皆既日食のタイミングを狙う
  └─ 久利を賢者の石製造へ導く
        ↓
  探索者にムナールの星石を外させる（③の罠）
        ↓
  時計塔で星の吸血鬼顕現 → 賢者の石完成 → ≪完全≫
        ↓
  久利教授の肉体経由でイゴーロナク顕現（⑦）
```

### ① 久利教授の操り方

- **完全乗っ取りではなく「悪意を通した操作」**（原文冒頭）
- 久利は表向き「ただの化学教授」。錬金術ゼミで賢者の石に固執
- 久利本人の認識：人間の血を媒介に、日食の日に星の吸血鬼をフラスコとして賢者の石を作る
- 被害現場の**五芒星はダミー**（久利は「魂を代償とした錬成の想起用」と思っている）→ 探索者・警察の注意を逸らす

### ② ネット魔導書と被害者トラップ（作者正本）

久利（イゴーロナクに操られ）が5chに載せた**グラーキの黙示録十二巻のページ写し**を見た者は、精神汚染を受ける。本人は自覚しにくい。**無意識のうちに操られ**、結界の**五つの頂点**──半世紀前に張られた五芒星結界の**古き印の起点**──へ向かう。

各頂点は結界の外縁（一松の街路）にある。そこで**星の吸血鬼**が待ち、操られた者を襲う。心臓の血を吸い取られて死ぬ。

| イゴーロナク側の成果 | 内容 |
|----------------------|------|
| **材料** | 賢者の石の錬成に必要な**人間の血** |
| **結界弱化** | 頂点を**死で汚す**ことで古き印の力を削り、結界全体を弱める |

現場に刻まれる**逆五芒星**は、久利には「探索者・警察の注意を逸らすダミー」に見えるが、結界弱化の魔術でもある。

| 運用メモ | 内容 |
|----------|------|
| 赤霧市内の人間 | イゴーロナクの影響を強く受け、操られやすい |
| 噂の急上昇 | 好奇心旺盛な人間を釣る |
| チェス隠しHP | 「頭のいい奴だけ」が踏む罠。写し閲覧で汚染 |

導入でも「賢者の石の作り方の掲示板」「協力者募集」など、**噂に乗じて人を集める**描写がある。

### ③ 結界を崩す手順

1. **結界の範囲外**で星の吸血鬼による殺人（吸血鬼事件・5名の犠牲者）
2. 現場に**逆五芒星**の魔法円 → 結界の影響力を弱める
3. **皆既日食**（クライマックス②で月食開始）の時間帯を狙う
4. **ムナールの星石を魔方陣から外させる**（③の罠）
   - 久利は「脅威は取り除かれた」と言い、探索者の協力を装う
   - 汚染された探索者は、POW失敗で勝手に星石を蹴とばそうとすることもある

### ④ 賢者の石と顕現の関係

- 星の吸血鬼に血を吸わせ → 水銀（純物質）→ 黒・白・赤の段階 → **賢者の石完成**
- 久利が「パーフェクション！」（≪完全≫）を唱え、完全体に変身
- 完全体久利を倒し首が落ちる → **イゴーロナクが顕現**（⑦）
- 賢者の石は術法増幅器（POW2倍・MP肩代わり）として、顕現・退散フェーズにも影響

### ⑤ 失敗時・エンド後

- 退散呪文成功 → 扉の中に引きずり込まれる
- 結界再起動（⑨）：ムナールの星石＋賢者の石を媒体に再発動。**賢者の石は消費される**
- エンド：久利教授の手記が渡る。「皆の知るところの秘密」扱い

### 久利とイゴーロナクの関係（整理）

| 久利教授が思っていること | 実際（イゴーロナク側） |
|--------------------------|------------------------|
| 自分は賢者の石の研究をしている | 久利は**器**。悪意で操られている |
| 五芒星は探索者へのダミー | 結界弱化の布石の一部 |
| ムナールの星石を外すのは「脅威除去」 | **結界の核を外す**ことが目的 |
| 賢者の石で肉体を完成させた | ≪完全≫後、首落下で**邪神が顕現** |

## 背景・動機（原文・参照用）

### 計画の時系列

1. 結界の範囲外で使役する**星の吸血鬼**による殺人事件（吸血鬼事件）
2. **逆五芒星**の魔法円で結界の影響力を弱める

### 賢者の石の製法（裏・原文）

```
① 星の吸血鬼に必要な分の人間の血を吸わせ、代謝させて水銀（純物質）を作る
② 皆既日食の時間に星の吸血鬼を召喚する（黒）
③ イブン＝グハジの粉をかける（白）（硫黄）
④ 血を吸わせる（赤）（プネウマ）
```

- 邪神の眷属は移動不可
- 精神支配の効果が弱まる
- 邪神の行動成功率は**半分**

#### 戦闘ルール（原文）

- 回避しない。1ラウンドに1度近接格闘による**受け流し**
- **久利教授への攻撃は触手でかばう**
- 魔術強化により、**吸血に成功するまで倒されない**
- **ムナールの星石**を持った者は攻撃しない。星石を押し当てると装甲無視の **1D6** ダメージ
- 毎ラウンド3回、近接格闘マヌーバーで鉤爪捕獲。成功→次ラウンドまでSTR対抗で引きはがさない限り噛みつき
- 噛みつき成功→星の吸血鬼の前身が赤く染まり、賢者の石製造が最終段階へ

**顕現条件**: 月光が完全に消え暗闇が訪れた後。クスクス声の方向に「イブン＝グハジの粉」をかける（探索者がやらなければ教授にやらせる）。

### エンド処理（⑧〜⑨）

**結界再起動中**（ムナールの星石を持っていない、魔導書を見たPC）:

- 頭痛。POWロール。失敗→毎ラウンド **10INT** と **10POW** 喪失（結界再起動まで続く）
- INTまたはPOWが0→**賢者の石を奪おうとする**（入手した場合、第2のイゴーロナクとなる**かもしれない**）

**結界再起動（⑨）**:

- 魔方陣の英語: 満月の夜に術を拡張／ムナールの星石で古き印の結界を貼るには**5人分のPOW**を捧げる
- ムナールの星石を中心に置き、賢者の石を乗せることで結界発動
